Artist and lead developer at Visualize Value, working alongside Jack Butcher on Checks and Opepen Edition. Lead developer of Opepen, a public art protocol on Ethereum with 16,000 tokens where creators submit interpretations and collectors vote on permanent collection pieces. Created multiple Opepen sets including Set 008 (exploring infinity) and Set 009 "Unity" (which hit 13,000% demand). Also crafted Snowpepen, 80 editions using original Chrome Squiggle code, praised by Snowfro.

Founder of PunkScape/Scapes, an NFT project that pioneered ENS domain integration and moved fully on-chain to ensure permanence. Author of "Understanding CryptoPunks," a comprehensive guide published with Visualize Value covering everything from art to smart contract code. Maintains GitHub repositories for Visualize Value projects including mint protocol, checks-contracts, and opepen.
